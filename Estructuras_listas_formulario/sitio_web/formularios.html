Etiqueta adicionales para la captura de datos ¡Excelente encontrarte por acá nuevamente! En este apartado descubriremos algunos casos especiales para la captura de datos dentro de un formulario. Contraseña Para las contraseñas usamos la etiqueta con el atributo type="password", lo que hará que no se muestren los caracteres cuando el usuario escribe. Contraseña Checkbox El checkbox es una caja le permite al usuario activar una opción o no. Para crear un checkbox empleamos la etiqueta con el atributo type="checkbox". Es importante colocar una etiqueta (label) y enlazarla con el atributo for al checkbox a través de su id, donde el valor del for y el valor del id deben coincidir.
